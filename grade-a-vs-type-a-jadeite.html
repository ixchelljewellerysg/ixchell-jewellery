Grade A Jade vs Type A Jadeite

Most people search for “Grade A jade.”
But in real gemology, this term is not correct.

The correct term is Type A jadeite — and understanding this difference is the first step towards recognising jadeite of peerless beauty.

Simple truth:
Grade A jade = commercial wording
Type A jadeite = real, natural jadeite

What is Type A Jadeite?

Type A jadeite refers to natural jadeite that has not been chemically treated.

It is:

  • Not acid washed
  • Not resin filled
  • Not artificially dyed

It is the only form of jadeite accepted as natural in gemological reports.

What is “Grade A Jade”?

“Grade A jade” is a marketing term.

It is commonly used online, especially in marketplaces, but it is not used in official lab reports.

This creates confusion — and often misleads buyers into thinking all “Grade A jade” is genuine.

Real certification uses:
Type A / Type B / Type C

Not Grade A / B / C

Type A vs Type B vs Type C

Type A Jadeite
Natural, untreated — highest value

Type B Jadeite
Chemically treated and resin-filled

Type C Jadeite
Dyed to enhance colour

Only Type A jadeite retains true structure, longevity and collector value.

Why This Matters

Many first-time buyers unknowingly purchase treated jade.

Because visually, treated jade can appear:

  • More vibrant
  • More uniform
  • More “perfect”

But over time:

  • Colour may fade
  • Structure weakens
  • Value drops significantly

Why Burmese Jadeite is the Benchmark

High-quality jadeite comes almost exclusively from Myanmar (Burma).

This is why “Burmese jadeite” is considered the global benchmark — not just for rarity, but for structure, translucency and depth.

How to Buy Jadeite Properly

If you are buying jadeite in Singapore or anywhere else, focus on:

  • Type (must be Type A)
  • Structure
  • Translucency
  • Natural colour

Not just brightness — but life within the stone.
